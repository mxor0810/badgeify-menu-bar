## Badgeify 是什么

说白了就是：**把任何 App 的图标放进菜单栏，并且实时显示未读消息数量**。

macOS 原生菜单栏只有系统级 App 和部分第三方工具会自己驻留进去。像 Mail、Messages、Slack、Discord 这类常用的通讯类 App，正常情况下你得盯着 Dock 才能看到消息角标——而很多人习惯把 Dock 隐藏，或者切换到其他虚拟桌面时根本看不到 Dock。

Badgeify 一键解决这个断层：把你想要监控的 App 图标拖进来，之后菜单栏就会持续显示它的未读数，点一下直接跳转打开 App。

## 核心功能有哪些

**消息角标实时同步**
只要 macOS 通知权限开着，Badgeify 就能读取 App 的未读数并在菜单栏实时更新。Mail、Messages 这类原生 App 效果最好，Slack、Discord 等第三方应用也支持。

**图标分组管理**
菜单栏空间寸土寸金，Badgeify 支持把多个 App 图标合并成一个目录（文件夹），点开才展开。不想让菜单栏堆满图标的人，这个功能很贴心。

**一键快启 / 切换 App**
点击菜单栏图标不只是"看看消息数"，还可以直接开启或切换到那个 App。对于 Dock 常年隐藏的用户来说，菜单栏变成了另一个启动台。

**智能隐藏 / 淡化**
没有未读消息时，可以设置图标自动透明或直接隐藏，等有消息了才出现。不用的时候不占位，有消息时才提醒你——这个设计挺细心的。

**个性化图标**
支持上传自定义 SVG 图标、调整强调色、自定义排序，能适配深色 / 浅色模式。有些用户甚至会给自己的 App 换成统一风格的图标，菜单栏看起来整洁很多。
